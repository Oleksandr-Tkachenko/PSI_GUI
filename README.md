# Private Set Intersection (PSI)
###GUI###
---
__Install:__
```
./configure.sh
```
```
sudo make install
```
__Clean:__ 
```
sudo make clean
```

__Remove:__ 
```
sudo make uninstall
```


###Dependencies:
---
* qt4-dev-tools
* lpsi-util
* psi-naive-hashing
* psi-cuckoo-hashing
* psi-intersection
* psi-simple-hashing
